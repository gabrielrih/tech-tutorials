# dump

Running dump on a specific database on Atlas:
```sh
mongodump --uri="mongodb+srv://username:password@my-cluster-pl-1.ayaki.mongodb.net/" --db database-name
```

> The backup is generated on a dump folder


# restore

Restoring a database from a bson backup (generated by mongodump command):
```sh
mongorestore --uri="mongodb+srv://username:password@my-cluster-pl-1.ayaki.mongodb.net/" --nsInclude database-name.* ./dump/
```

> It gets all the collections inside _./dump/_ and restores it on the _database-name_ database.
